# 剪貼簿應用程式

這是一個基於 Vue.js 的剪貼簿應用程式，提供兩種輸入模式讓使用者可以快速新增、儲存和管理內容。應用程式支援自定義背景顏色並使用 `localStorage` 儲存資料，以便下次開啟時保持資料狀態。

## 功能

- **雙模式**：支援多行模式與單行模式，可切換不同的輸入模式以適應不同需求。
- **即時儲存**：資料會自動保存至 `localStorage`，重新開啟時仍可顯示上次儲存的內容。
- **背景顏色自定義**：使用者可以選擇背景顏色，並即時應用至整個應用程式。
- **複製功能**：單擊複製按鈕，即可將內容複製到剪貼簿。
- **刪除功能**：可刪除不需要的項目。
- **標題寬度調整**：在單行模式中，使用者可調整標題欄的寬度。

## 使用說明

1. **切換模式**：點擊右上方的模式按鈕切換多行模式或單行模式。
2. **新增內容**：
   - **多行模式**：在多行文字區域輸入內容，點擊其他區域即可自動新增。
   - **單行模式**：在單行輸入框內輸入內容，按 Enter 或點擊其他區域自動新增。
3. **刪除項目**：點擊刪除按鈕即可移除對應項目。
4. **複製內容**：點擊複製按鈕可將對應項目複製到剪貼簿。
5. **調整背景顏色**：左上方顏色選擇器用於改變應用程式的背景顏色。
6. **調整標題寬度**（僅單行模式）：在單行模式中，可用拖拉滑桿調整標題欄寬度。
